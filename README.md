Este projeto marca o meu primeiro contato prático com o desenvolvimento mobile utilizando o framework **Flutter** e a linguagem **Dart**. Foi desenvolvido como parte de uma atividade prática da graduação em **Análise e Desenvolvimento de Sistemas** no Centro Universitário Maurício de Nassau.

⚠️ **NOTA DE ESCOPO:** Este é um projeto estritamente acadêmico, desenvolvido apenas para **apresentação em sala de aula e fixação de conceitos**. O conteúdo deste repositório serve como evidência do meu aprendizado inicial na tecnologia e não possui planos para ser publicado em lojas de aplicativos (Google Play ou App Store).

## 📌 Funcionalidades e Objetivos
Sendo um projeto introdutório, o foco principal foi entender a árvore de Widgets do Flutter, a navegação básica e a construção de interfaces. O aplicativo demonstra:
* **Estrutura de Telas:** Entendimento do roteamento e transição entre páginas.
* **Desenvolvimento de UI:** Criação de telas e componentes visuais, com destaque para a interface desenvolvida em `TelaDeContato.dart`.
* **Fundamentos de Widgets:** Aplicação prática da diferença e uso de Widgets Stateless e Stateful.

## 🛠️ Tecnologias Utilizadas
- **Framework:** Flutter
- **Linguagem:** Dart
- **Paradigma:** Desenvolvimento de Interface Declarativa
